# 智慧图书管理系统 java1170

## 项目概述

智慧图书管理系统是基于Spring Boot和Vue的现代化图书管理系统。系统实现了前后端分离，支持书籍信息的全面管理以及用户借阅流程的便捷操作。

## 技术栈

- **后端：** Spring Boot，搭配MyBatis Plus和MySQL
- **前端：** Vue 2，使用HTML进行页面构建
- **适配环境：** Java 1.8

## 系统角色与功能模块

### 管理员

- **书籍信息管理：** 增删改查操作
- **用户账号管理：** 管理用户账户
- **借阅记录管理：** 查看与统计借阅记录

### 用户

- **书籍查询：** 快速查找图书
- **借阅操作：** 简便的借阅流程
- **归还操作：** 归还图书
- **个人中心：** 查看借阅历史和到期提醒

## 运行环境

- **后端：** 需部署在支持Java 1.8及以上版本的Servlet容器上
- **前端：** 需在现代浏览器上运行
- **数据库：** MySQL数据库服务器

## 部署步骤

1. 准备Java 1.8及以上版本的运行环境。
2. 配置MySQL数据库，导入初始数据。
3. 编译后端代码，并部署到Servlet容器。
4. 构建前端代码，并部署到Web服务器。
